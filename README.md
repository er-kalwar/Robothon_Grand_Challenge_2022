WHAT IS ROBOTHON?

  Robothon is an international competition and benchmarking event for measuring state-of-the-art performance in robot manipulation. Teams compete using a robot platform to autonomously execute a series of hand-picked manual tasks inspired by real world industry challenges across multiple sites.
